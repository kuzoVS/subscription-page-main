# VPN Subscription Page - Minimalist Dark Design

## Color Scheme
- **Background**: Pure black (#000000)
- **Primary containers**: Dark gray (#111111)
- **Borders**: Lighter gray (#222222)
- **Hover states**: Medium gray (#333333)
- **Icons**: Very dark gray (#1a1a1a) with gray borders (#333333)
- **Text**: White (#ffffff) and medium gray (#888888)

## Layout Structure
- Clean, spacious layout with ample whitespace
- Minimalist header with simple branding
- Subscription information displayed in clean card format
- Responsive grid for information display

## Header Design
- Solid dark background (#0a0a0a)
- Thin bottom border (#222222)
- Clean logo and title in white
- No animations or effects

## Subscription Information Cards
- Square/rectangular cards with rounded corners (small radius)
- Dark background (#111111) with thin border (#222222)
- Clean icon containers in dark gray (#1a1a1a) with border (#333333)
- Minimal hover effect (border color changes to #333333)
- Clear typography hierarchy

## Typography
- Clean, modern font stack
- White text for primary information
- Gray text (#888888) for secondary information
- Minimal font weights (400 for labels, 500 for values)
- Consistent sizing and spacing
- **All button labels must use sentence case (lowercase), never ALL CAPS**

## Buttons
- Pill shape with border-radius: 20px
- Dark background (#1a1a1a) with thin border (#333333)
- Sentence case text (e.g. "Windows (установщик)", "Добавить подписку")
- No uppercase or all-caps styling anywhere
- Small icon inline with text where applicable

## OS Selector (Windows dropdown)
- Dark neutral background (#1a1a1a), thin border (#333333)
- No bright cyan or accent colors
- Same style as other secondary buttons
- Small Windows icon + label + dropdown arrow

## App Tabs (Koala Clash / FClashX / Clash Verge)
- Simple pill-shaped tabs, no emoji or mascot icons
- Active tab: white background (#ffffff) with black text (#000000)
- Inactive tabs: dark background (#1a1a1a), gray text (#888888), thin border (#222222)
- No animal icons, no decorative images inside tabs

## Installation Steps
- Each step has an icon on the left and text on the right
- All step icons must use the same visual style: dark background (#1a1a1a), thin gray border (#333333), simple line icon in gray (#888888)
- No colored or accent icons (remove red/orange warning icon — use same neutral gray style)
- A thin vertical connector line (#222222) runs between step icons to visually link them
- Step titles in white (#ffffff), descriptions in gray (#888888)

## Interactions
- Subtle hover effects (border color change only)
- No animations or transitions (or very minimal)
- Clean focus states with thin outline

## Scrollbar
- Minimal 4px wide scrollbar
- Dark gray thumb (#333333)
- Black track background

## Overall Aesthetic
- Extreme minimalism with no unnecessary elements
- Monochromatic black/gray/white color palette
- Clean geometric shapes
- Focus on functionality over decoration
- Professional, secure feeling appropriate for VPN service
